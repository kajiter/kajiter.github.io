iOS开发快捷方式

Xcode7.0以上打开HTTP权限

以Open as ->Source Code 的方法打开info.plist文件，然后加入以下代码
这段代码放的位置不用固定

<key>NSAppTransportSecurity</key>
<dict>
    <key>NSAllowsArbitraryLoads</key>
    <true/>
</dict>

  pod 'AFNetworking'
  pod 'FMDB'
  pod 'MJRefresh'
  pod 'ShareSDK3'

end
