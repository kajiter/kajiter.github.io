HTML (Hyper Text Markup Language) 超文本标记语言

Hyper Text

超文本是一种信息组织方式。使用超链接（URL）的方式，将本地某一文件中的信息（包括：文字，图表，图片，声音，等）与其他文件 或是 地理位置上很远的某台计算机上的文件信息，随机的链接起来，为人们查找、检索提供方便。

HTML

超文本标记语言是通过一系列、对应的标签对（如：<html> ··· </html>） ，以及文本信息，所组成的。
