全新超视网膜显示屏，精工细造，精美绽放。

超视网膜高清显示屏采用的 OLED 显示面板，远非普通的 OLED。它按照我们的严苛标准量身打造，拥有 1,000,000:1 的对比度、高分辨率和出色的亮度、广色域支持，以及出众的色彩精准度，能够实现 HDR 显示。

亮眼之处，遍及每个边沿。

在设计这块 OLED 显示面板时，我们采用了创新的折叠和电路堆叠技术，使它能够贴合机身曲线，一直延伸至四角。然后，使用一项称为子像素抗锯齿处理的技术调整每颗像素，确保边缘显示平滑无变形。

iPhone 上迄今
最精准的色彩显示。

超视网膜高清显示屏的色彩管理系统更进一步。无论显示的内容使用的是 P3 还是 sRGB 色彩模式，iPhone X 都能自动以对应的格式显示。因此，创作者们想要表达的色彩，都能准确地传递到你的眼中。

原彩显示，
什么都看得更舒服。

原彩显示技术使用先进的六通道环境光传感器，能够根据周围环境光的色温来精细调节屏幕的白平衡，让屏幕上显示的图像看来就像在印刷品上一样自然，眼睛就不会那么容易疲劳了。

HDR
在 iPhone 上的首映，
现已开场。

iPhone X 实现了真正的 HDR (高动态范围) 显示，让你可以欣赏到符合杜比视界和 HDR10 标准的影片和节目，HDR 照片的显示效果也更加惊艳。另外，你还能观看其他视频网站提供的精彩 HDR 内容。

iPhone 上迄今
最坚固耐用的玻璃面板。

机身前后面板都是全玻璃设计，采用了 iPhone 上迄今最坚固耐用的玻璃材料，其强化层比以往增厚了 50% 之多。我们运用七层染色工艺，精准控制色调和不透明度，并加入反射光学层来增强色彩。另外，玻璃面板还覆有防油渍涂层，让你能轻易抹去污迹和指纹。

无线世界，无线充电。

我们一直希望 iPhone 能摆脱充电线或耳机线的束缚，成为一部真正的无线设备。iPhone X 的玻璃背板和内置的无线充电系统，正是为将来的无线世界而设计3。

独家打造的手术级不锈钢。

iPhone X 机身侧边环绕着起加固作用的不锈钢边框，它采用 Apple 特别设计的合金材料，不仅坚固耐用、质地纯净，而且经过精心打磨，异常精美。在深空灰色外观的机型上，我们还运用了一项被称为“物理气相沉积”的工艺，确保不锈钢边框颜色与玻璃颜色精准一致。

防溅抗水，还防尘。

我们深入微观层面对 iPhone X 进行了精密设计，以实现防溅、抗水和防尘性能2。

你的脸，现在就是你的密码。面容 ID 功能为设备解锁、身份验证和支付带来了一种安全、私密的新方式。

面容 ID

有什么动作能比轻触一下还要自然？答案是：看一眼。这就是面容 ID 的灵感来源。面容 ID 是一套强大、安全的验证系统，用起来甚至比触控 ID 还要方便。通过它解锁设备或进行支付，都十分快速、轻松而自然。

让人惊叹的科技，
造就让人赞叹的简单。

原深感摄像头系统集多项创新技术于一身。这些技术实时协作，可以通过你脸部的精细深度图在瞬间认出你。

点阵投影器

通过将 30,000 多个肉眼不可见的光点投影在你脸部，绘制出你独一无二的面谱。

红外镜头

红外镜头会读取点阵图案，捕捉它的红外图像，然后将数据发送至 A11 仿生这款芯片中的安全隔区，以确认是否匹配。

泛光感应元件

借助不可见的红外光线，即使在黑暗中也能识别你的脸。

你的脸，
就是一张安全的通行证。

有了面容 ID 的帮助，iPhone X 只有在你注视它时才会解锁。经过精心设计，它还能防止被照片或面具欺骗。你的面容 ID 信息会被加密，并在安全隔区中受到保护。而且数据不会传出你的设备，也绝不会备份到 iCloud 或其他任何地方，以此确保私密性。

一旦认识你，
就总能认出你。

面容 ID 利用先进的机器学习识别你样貌的变化。即使你戴着帽子，留起胡须，或者佩戴眼镜，甚至是不同款式的墨镜，你也一样可以使用面容 ID。因此，就算朋友们一眼认不出你，面容 ID 也能认出你。

善解人意，
在你需要时才启动。

从进一步的安全层面考虑，我们为面容 ID 增加了注视感知功能，也就是说，只有当你睁眼注视着 iPhone X 时，它才会解锁。另外，面容 ID 功能也可用来显示通知和信息，在你阅读内容时保持屏幕常亮，或调低闹钟或铃声的音量。

A11 仿生

能否造出一枚既有强劲动力，又具备高度智能的芯片？这就是 A11 仿生：iPhone 有史以来最强大、最智能的芯片。

超乎想象的
强劲动力。

A11 仿生拥有六个核心和 43 亿个晶体管，与 A10 Fusion 芯片相比，四个能效核心的速度提升最高可达 70%，两个性能核心的速度提升最高可达 25% 之多。当需要提升处理速度时，中央处理器甚至能同时发挥全部六个核心的性能。

什么是
神经网络引擎？

有一种人工智能叫作机器学习，即让电脑通过观察的方式进行学习。而神经网络引擎就是专为机器学习而开发的硬件，它不仅能执行神经网络所需的高速运算，而且具有杰出的能效。

神经网络引擎
有什么作用？

A11 仿生中的神经网络引擎采用双核设计，能够识别人物、地点和物体。它的每秒运算次数最高可达 6000 亿次，能轻松胜任机器学习任务，并为面容 ID 和“动话表情”等创新的功能提供强大的动力。

为高能效打造。

A11 仿生会在性能核心与能效核心之间合理分配任务，智能地管理芯片的澎湃性能。因此，无论是发短信还是浏览网页，许多你常做的事情都能迅速完成，而且耗电也更少。再加上量身定制的电池设计，猜猜会有什么惊喜？那就是 iPhone X 充满电后的电池续航时间，与 iPhone 7 相比最多可延长两个小时4。

增强现实，再增强一步。

得益于 A11 仿生这款芯片，增强现实类游戏和 app 的流畅度、真实感都达到了一个新的高度。
